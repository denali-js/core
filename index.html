The best way to build robust APIs in Node

Denali is an opinionated Node.js framework that lets you focus on shipping APIs,
instead of bikeshedding about the plumbing.
 
 
 
          $ npm install -g denali
$ denali new my-api
$ cd my-api && denali serve

Heads up!

Denali is still in alpha. We don't recommend using it for production apps just yet. Check out the road to beta to see what's left!

API-first

You won't find any templating languages, views, or other HTML rendering support out of the box. Denali is unapologetically focused on APIs first. First class support for JSON serializers, RESTful routing, and powerful testing helpers make building APIs easier than ever.

Developer happiness

Denali ships with a top-notch CLI which can run reload-on-change development server, scaffold new code, run tests, and more. Extensibility is baked into the core as well, with Addons providing a powerful approach to incorporating third party code.

Data Flexibility

Thanks to it's unique model + adapter approach, Denali let's you use whatever ORM you want with it. And Denali Addons will work with it out of the box. You can even use multiple ORMs in the same project to fully leverage the strengths of each data store (a.k.a polyglot persistence).
